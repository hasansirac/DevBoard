# DevBoard

Modern task and board management application.

## Features

- User register & login
- JWT authentication
- Create boards
- Create tasks
- Task priority & status
- Modern responsive UI
- REST API backend

## Technologies

### Backend

- Node.js
- Express.js
- SQLite
- JWT

### Frontend

- HTML
- CSS
- JavaScript

## Installation

```bash
npm install
```

## Run Backend

```bash
npm start
```

## Run Frontend

Open:

```text
frontend/index.html
```

with Live Server.
